# 🍔 Munchly - Food Delivery System

## Overview

Munchly is a full-stack food delivery web application developed using React, Spring Boot, and MySQL. The application allows users to browse food items, manage carts, place orders, and enables administrators to manage menu items and orders.

## Features

* User Registration and Login
* Food Menu Browsing
* Add to Cart Functionality
* Order Placement
* Admin Dashboard
* Food Item Management
* Order Management
* Image Upload Support

## Technology Stack

### Frontend

* React.js
* Vite
* JavaScript
* CSS

### Backend

* Spring Boot
* Java
* REST APIs

### Database

* MySQL
